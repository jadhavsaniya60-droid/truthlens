TruthLens 🔍

TruthLens is a Python-based fake news detection project.

It analyzes news headlines or text and predicts whether the content is real or fake.

 🚀 Features
- Takes user input (news headline/text)
- Applies basic detection logic
- Predicts Real or Fake

🛠 Tech Stack
- Python
